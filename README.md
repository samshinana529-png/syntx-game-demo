# 👑 Crown Quest

## Overview
Crown Quest is an educational board game developed by Syntax Rangers. Players roll a dice, move across the board, answer questions, collect points, and race to reach the Crown Tile before their opponents.

The game combines entertainment and learning by encouraging players to answer educational questions while progressing through the board.

---

## Features

- Interactive board game interface
- Dice rolling system
- Multiple player support
- Question and answer challenges
- Score tracking
- Winner detection
- Educational gameplay
- Browser-based gameplay

---

## How to Play

### Objective
Be the first player to reach the Crown Tile while earning points through correct answers.

### Rules

1. Each player starts at the Base Tile.
2. Players take turns rolling the dice.
3. The number rolled determines how many tiles the player moves.
4. When landing on a question tile, the player must answer the question.
5. Correct answers earn points.
6. Incorrect answers may result in losing a turn or no points being awarded.
7. The first player to reach the Crown Tile wins the game.

---

## Controls

| Action | Description |
|----------|------------|
| Roll Dice | Move player forward |
| Answer Question | Earn points |
| Next Turn | Pass turn to next player |

## Installation

### Option 1: Play Locally

1. Download the project files.
2. Extract the ZIP folder.
3. Open the project folder.
4. Double-click `index.html`.

The game will launch in your web browser.

---

### Option 2: Run Using VS Code

1. Open the project folder in Visual Studio Code.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.
